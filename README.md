### :information_source: How To Run

#### Start Server

###### It is recommended to run the server in a separated window.

```bash

$ cd scalable-capital-ts

# install dependencies
$ npm install

# Run server
$ npm run server
```

#### Run Tests

##### Enzyme

```bash

$ cd scalable-capital-ts

# install dependencies
$ npm install

# Run tests
$ npm run test
```

##### Cypress

```bash

$ cd scalable-capital-ts

# install dependencies
$ npm install

# Run tests with UI
$ npm run cypress:open

# Run tests in background
$ npm run cypress:run
```

#### Run Application

```bash

$ cd scalable-capital-ts

# install dependencies
$ npm install

# Run application
$ npm start
```
